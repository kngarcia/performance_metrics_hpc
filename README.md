# Taller de Procesamiento de Imágenes con CUDA: Múltiples Filtros
Contiene la implementación de varios filtros y operaciones de procesamiento de imágenes aceleradas utilizando NVIDIA CUDA a través de la biblioteca `numba.cuda` en Python. El objetivo de este taller es demostrar las ventajas de la paralelización en GPU para tareas computacionalmente intensivas en el ámbito del procesamiento de imágenes.
